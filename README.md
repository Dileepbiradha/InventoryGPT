# InventoryGPT

AI Powered Inventory Management System

## Features

- Product Management
- Inventory Tracking
- Stock In / Stock Out
- Purchase Orders
- Suppliers Management
- Customers Management
- Analytics Dashboard
- Low Stock Alerts
- AI Assistant

## Tech Stack

### Frontend
- React
- Vite
- Axios
- Chart.js

### Backend
- Flask
- SQLAlchemy
- JWT Authentication

### Database
- SQLite

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
flask run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
